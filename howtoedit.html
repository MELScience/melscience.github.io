How to Edit An Experiment

This website is hosted at GitHub. You can propose any change to the website by forking the website repository, adding/editing

1. Create a GitHub account

If you already have a GitHub account you can skip this step. Go to GitHub.com and create a new account.

2. Click the "Edit" link on the page you want to edit

You can edit data pages for experiments, reagents and categories. For example if you want to edit Preparation of hydrogen using hydrochloric acid and zinc experiment page you need to open it, then click the Edit link on top of the page.

3. Create your own copy of the website by forking the repository

Press the Fork this repository and propose changes button to create your own copy of the website. That is how GitHub works - you create your own copy, edit it and than propose a change back to the main website.

4. Edit the page

Edit the page. Each experiment page is a text file with .html extension that contains YAML description of the experiment.

5. Propose file change

After you are done with editing add description of what was changed on the bottom of the page and press the "Propose file change" button and then "Create pull request" button on the appeared page.

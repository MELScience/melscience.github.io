How to Add A New Experiment

1. Create a GitHub account

If you already have a GitHub account you can skip this step. Go to GitHub.com and create a new account. Make sure you have your email addres verified - GitHub will send you an email to verify your email address and you should click a link in this email.

2. Fork MEL Science website repository

Open the MEL Science website repository at https://github.com/MELScience/melscience.github.io/ and click the Fork button on the top. It will create your own copy of the website (such copies called forks in GitHub). Since now on you will edit your own copy and only after you have made all the needed changes and test it you will add the changes to the main website.

3. Create "gh-pages" branch

Before editing the website you need to make one more technical step: create a new branch called "gh-pages". It is needed to tell GitHub to handle this repository as a website which will be helpful for you in order to test how your changes look like on the website. Click the branch: master combobox and enter gh-pages in the appeared edit field to create a new branch. Please note that it is important that create branch has exactly this name.

4. Open the "experiments" folder

After you have forked the website repository and created "gh-pages" branch open the "experiments" folder.

5. Create a new experiment file

Click the "+" link on the top of the "experiment" folder page.

6. Choose experiment file name

The experiment file name should be descriptive, all small letters, use "-" symbol between the words and end with ".html" file extension. Here are samples of good experiment file names:

  • ammonium-chloride-smoke.html
  • burning-of-sugar-cube-with-ash-as-a-catalyst.html
  • catalytic-oxidation-of-acetone-on-copper-wire.html
  • flameless-burning-of-hexamine-fuel-tablet-with-cr2o3-as-a-catalyst.html
  • preparation-of-hydrogen-using-hydrochloric-acid-and-zinc.html

7. Copy sample experiment text to the file

Copy the following sample experiment text to your just created file 

---
id: preparation-of-hydrogen-using-aluminium-foil-and-sodium-hydroxide
type: experiment
experiment_name: [sample experiment name]
category: gases
reaction: [reaction formula here]
description: |
  [description here]
explosion: 1
color_change: 0
fire: 1
precipitate: 0
gassing: 1
smoke: 0
catalyst: 0
phases: liquid+solid+gas
trick: 
comments: 
safety: |
  [safety description here] 
youtube_videos:
  - link: http://www.youtube.com/watch?v=mnDgGbo63fw
  - link: http://www.youtube.com/watch?v=LKGAfwC9EhU
links:
  - link: https://sites.google.com/site/chemlabchemistry/experiments-1/smoke-with-ammonia
    title: "Smoke with ammonia 2"
    language: en
  - link: http://mattson.creighton.edu/NH3/index.html
    title: "Experiment 7. Gaseous Ammonia Reacts with Gaseous Hydrogen Chloride"
    language: en
reagents: 
  - hydrochloric-acid-concentrated
  - ammonium-hydroxide-solution-30
layout: experiment
---

8. Update experiment id and experiment name in the file

Copy/paste your file name without ".html" file extension to id value (the first line). Edit the experiment_name (the third line).

9. Save your changes

Press the Commit new file button on the bottom of the page.

10. Test the first draft version

Now a new experiment page is created. You can test it online in your copy. Its link is: 

https://[your_login].github.io/melscience.github.io/experiments/[your_page_name].html

11. Edit experiment page

Now you are ready to actually add experiment information. Open the experiment page:

And click the edit icon just over the file text:

12. Edit: add links

Search for webpages that contain description of this experiment. Please include not more than 10 link and include only those webpages that contain experiment description not generic information about this reaction. For each link add the following three lines to the links section: 

  - link: [webpage_url_here]
    title: "[webpage title here]"
    language: en

Add pages for the languages you understand. Use 2-letter 639-1 language codes.

13. Edit: add YouTube videos

Search for YouTube videos about this experiment. For each video add one line to the youtube_videos section: 

  - link: http://www.youtube.com/watch?v=[youtube_video_id]

Please remove all unneeded parameters from YouTube URL. URL must be only in the form as above.

14. Edit: description

Add experiment description that contains information how to make this experiment and some science background. Please check other experiment descriptions for the style. You can use markdown in the description.

15. Edit: category

Select one of the following categories for the experiment and add it to the category field.

16. Edit: reaction formula

Put reaction formula into the reaction field. Please use the following notations:

  • Use ↑ symbol for appeared gases
  • Use ↓ symbol for precipitate
  • Use =[catalyst]= to show that this reaction require a catalyst, add catalyst name instead of catalyst (for example: =[Pt]=)

17. Edit: reagents

Check if the needed reagents are already described in the list of reagents. If not create new reagent pages for missed reagents. Add list of all used reagent ids into the reagents section.

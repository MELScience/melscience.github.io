--- title: How to Add A Reagent layout: edit ---

1. Create a GitHub account

If you already have a GitHub account you can skip this step. Go to GitHub.com and create a new account. Make sure you have your email addres verified - GitHub will send you an email to verify your email address and you should click a link in this email.

2. Fork MEL Science website repository

Open the MEL Science website repository at https://github.com/MELScience/melscience.github.io/ and click the Fork button on the top. It will create your own copy of the website (such copies called forks in GitHub). Since now on you will edit your own copy and only after you have made all the needed changes and test it you will add the changes to the main website.

3. Create "gh-pages" branch

Before editing the website you need to make one more technical step: create a new branch called "gh-pages". It is needed to tell GitHub to handle this repository as a website which will be helpful for you in order to test how your changes look like on the website. Click the branch: master combobox and enter gh-pages in the appeared edit field to create a new branch. Please note that it is important that create branch has exactly this name.

4. Open the "reagents" folder

After you have forked the website repository and created "gh-pages" branch open the "reagents" folder.

5. Create a new reagent file

Click the "+" link on the top of the "reagent" folder page.

6. Choose reagent file name

The reagent file name should be descriptive, all small letters, use "-" symbol between the words and end with ".html" file extension. If the reagent is a solution add information about concentration to its id. Here are samples of good reagent file names:

  • ammonium-hydroxide-solution-30.html
  • hexamine-fuel-tablet.html
  • hydrochloric-acid-concentrated.html
  • hydrochloric-acid-solution-10.html
  • steel-wool.html
  • tin-2-chloride.html

7. Copy sample reagent text into the file

Copy the following sample experiment text into your just created file 

---
id: [reagent-id]
type: reagent
reagent_name: [reagent name]
substance_name: [substance name]
formula: [substance formula]
form: [reagent form such as solution, powder, wire, granules, etc]
chemspider_id: [id in chemspider database]
wikipedia: [link to the wikipedia article about this reagent]
nfpa_health: [nfpa health index (0, 1, 2, 3 or 4)]
nfpa_flammability: [nfpa flammability index (0, 1, 2, 3 or 4)]
nfpa_reactivity: [nfpa reactivity index (0, 1, 2, 3 or 4)]
nfpa_special: ACID
risk_phrases: [R23, R35]
safity_phrases: [S1/2, S9, S26, S36/37/39, S45]
msds: 
  - [link to a MSDS page/file 1]
  - [link to a MSDS page/file 2]
  - [link to a MSDS page/file 3]
ld50: [median lethal dose in milligrams per killogram]  
hazard_symbol: [hazard symbol]
other_names:
  - [another reagent name 1]
  - [another reagent name 2]
layout: reagent
---

8. Update reagent id and reagent name in the file

Copy/paste your file name without ".html" file extension to id value (the first line). Edit the reagent_name (the third line). Try to make a describing reagent name. Reagent name should start with a small letter. If the reagent is a solution add information about concentration to its name.

9. Save your changes

Press the Commit new file button on the bottom of the page.

10. Test the first draft version

Now a new reagent page is created. You can test it online in your copy. Its link is: 

https://[your_login].github.io/melscience.github.io/reagents/[your_page_name].html

11. Edit reagent page

Now you are ready to actually add reagent information. Open the reagent page:

And click the edit icon just over the file text:

12. Edit: add Wikipedia link

Replace [link to the wikipedia article about this reagent] with Wikipedia link of the English article about this reagent.

13. Edit: add substance name

Sometimes a reagent is one particular form of a substance that is available in other forms. For example a reagent could be a solution of a particular solt, substance in that case would be name of the solt itself. Sometimes reagent_name and substance_name are the same.

Here are few examples:
reagent_name substance_name
acetone acetone
aluminium foil aluminium
ammonium hydroxide solution 30% ammonium hydroxide
barium chloride barium chloride
copper wire copper
hexamine fuel tablet hexamethylenetetramine
hydrochloric acid concentrated hydrogen chloride
hydrochloric acid solution 10% hydrogen chloride
malachite copper(II) carbonate basic
potassium sulfate potassium sulfate
steel wool iron

14. Edit: add molecular formula

Replace [substance formula] with reagent molecular formula.

15. Edit: add reagent form

The same substance could come in different forms. Sometimes it is important for an experiment in which form it is. We consider different forms as different reagents. Replace [reagent form such as solution, powder, wire, granules, etc] with one of the following forms:
Name Description
liquid If liquid is a natural form for this substance
solution
powder Solid substance in form of powder
foil
wire
granules
solid Other forms of solid substance

16. Edit: add ChemSpider ID

Find your reagent in ChemSpider database. Replace [id in chemspider database] with ChemSpider ID.

17. Edit: add links to MSDS pages

Search for MSDS pages for this reagent. Briefly read each page before adding it to make sure it is about the reagent you are adding. Add links to the msds sections.

17. Edit: add NFPA information

Chech MSDS pages you have found before to find NFPA 704 safety codes for the reagent. Find Flammability, Health, Instability/reactivity and Special notice and add them to the corresponding sections. If different sources provide different information be on the safe side.

nfpa_special should have one of the following values or empty:

    OX
  • W
  • SA
  • COR
  • BIO
  • CRYO
  • ACID
  • ALK
  • RA
  • WOX

18. Edit: add LD50 information

Chech MSDS pages you have found before to find LD50 information. Replace [median lethal dose in milligrams per killogram] with a number of milligrams of LD50 per killogram. If different MSDS sheets provide different numbers be on the safe side.

19. Edit: add risk phrases and safety phrases

Chech MSDS pages you have found before to find risk phrases and safety phrases for this reagent. See the examples below:

barium chloride: 

risk_phrases: [R20, R25]
safity_phrases: [S1/2, S45]

platinum wire: 

risk_phrases: []
safity_phrases: []

tin(II) chloride: 

risk_phrases: [R22, R34, R43]
safity_phrases: [S26, S36/37/39, S45]
